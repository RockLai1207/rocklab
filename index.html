Rock Lab 用 GitHub Gist 儲存資料，跨裝置同步。設定一次就好。

Step 1 開啟 github.com/settings/tokens/new
Step 2 Note 填 rocklab，Expiration 選 No expiration，往下勾選 gist，按 Generate token
Step 3 複製 token（ghp_ 開頭），貼到下方
